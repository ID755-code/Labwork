Basic Workstation Setup

Steps to Create Gmail Account

  • Visit Gmail: Go to the Gmail website (www.gmail.com).
  • Click on "Create account": You'll usually find this option near the bottom of the page, depending on the layout.
  • Fill out the form: Enter your first and last name, choose a username, and set a strong password.
  • Add and verify your phone number: This is optional but recommended for account recovery.
  • Add and verify your email address: Another optional step to help with account recovery.
  • Complete the CAPTCHA: Confirm you're a real person.
  • Agree to Terms of Service: Read and agree to Google's terms.
  • Your account is created: Now you can start using your new Gmail account.

Steps to Create GitHub Account

  • Visit GitHub: Go to the GitHub website (www.github.com).
  • Sign Up: Click the "Sign Up" button to start creating your account.
  • Enter Your Information: Provide a username, email address, and create a password.
  • Verify Your Email Address: Check your email for a verification link.
  • Complete the CAPTCHA: Confirm you're a real person.
  • Choose Your Plan: Select a plan; the free plan is usually sufficient.
  • Welcome to GitHub: Your account is ready for use.

Steps to Connect VS Code to GitHub

  • Install Git: Ensure Git is installed on your computer.
  • Install VS Code: Download and install Visual Studio Code.
  • Open VS Code: Launch the editor and open or create a project.
  • Initialize Git Repository: Use the terminal in VS Code to run git init.
  • Stage and Commit Changes: Use git add . and git commit -m "Initial commit".
  • Create a GitHub Repository: Log in to GitHub and create a new repository.
  • Link Local Repository: Use the command palette in VS Code to add the remote URL.
  • Push to GitHub: Use the command palette in VS Code to push your changes.
  • Verify on GitHub: Check your GitHub repository to confirm the changes were pushed successfully.

Steps for Cloning Repositories

  • Find the Repository on GitHub: Navigate to the repository you want to clone.
  • Copy the Repository URL: Click the "Code" button and copy the URL.
  • Clone Repository: Use the command palette in VS Code to clone the repository.
  • Open the Cloned Repository: VS Code will prompt you to open the repository after cloning.

Steps to Push Changes to GitHub

  • Make Changes in VS Code: Edit the files in your project.
  • Stage Changes: Use the Source Control view to stage changes.
  • Commit Changes: Add a commit message and commit your changes.
  • Push Changes to GitHub: Use the command palette to push your changes.
  • Verify on GitHub: Check your repository to ensure the changes were pushed.

Steps to Publish Repo as Live Site

  • Prepare Your Repository: Ensure your repository has all necessary files for a static website.
  • Enable GitHub Pages: Go to the repository settings and enable GitHub Pages from the appropriate branch.
  • Verify and Deploy: GitHub Pages will deploy your site; check the provided URL to access your live site.
