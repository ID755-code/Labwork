Setting Up XAMPP

  1. Download XAMPP

    Visit the official Apache Friends website at https://www.apachefriends.org and download the XAMPP installer for your operating system (Windows, macOS, Linux).

  2. Run the Installer

    After downloading, open the installer file (e.g., xampp-installer.exe on Windows). Follow the on-screen instructions to start the installation process.

  3. Select Components

    During the installation, you'll be asked to choose components. By default, Apache, MySQL, PHP, and phpMyAdmin are selected. Unless you have specific needs, you can leave these settings as they are.

  4. Choose Installation Directory

    Specify where you want to install XAMPP. The default directory is C:\xampp on Windows and /Applications/XAMPP on macOS. You can change this directory if desired.

  5. Complete the Installation

    Click "Next" or "Install" to continue. Wait for the installation to finish, which might take a few minutes depending on your system.

  6. Start XAMPP

    After installation, open the XAMPP Control Panel. On Windows, you can find it in the Start menu or on the desktop. On macOS, itâ€™s in the Applications folder. Start the Apache and MySQL modules by clicking the "Start" button next to each.

  7. Verify the Installation

    Open your web browser and type http://localhost in the address bar. If everything is set up correctly, you should see the XAMPP dashboard, confirming that Apache and MySQL are running.

Follow these steps, and you'll have XAMPP successfully set up on your computer, ready for web development.
