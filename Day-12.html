SQL Logical and String Functions

Logical Operations

  • **AND**: Retrieves rows where both conditions are true. 
    SELECT * FROM EMPLOYEES WHERE SALARY < 1000 AND AGE > 25;
  • **OR**: Retrieves rows where at least one of the conditions is true. 
    SELECT * FROM STUDENTS WHERE RANK > 20 OR MARK > 600;
  • **NOT**: Retrieves rows where the condition is false. 
    SELECT * FROM EMPLOYEES WHERE NOT SALARY = 10000;

GROUP Operations

  • **GROUP BY**: Groups rows that have the same values in specified columns. 
    SELECT FName, LName FROM EMPLOYEES GROUP BY Name;

Aggregate Functions (Mathematical Operations)

  • **AVERAGE**: Calculates the average value of a numeric column. 
    SELECT AVG(salary) FROM EMPLOYEES;
  • **MINIMUM**: Retrieves the minimum value in a column. 
    SELECT MIN(salary) FROM EMPLOYEES;
  • **MAXIMUM**: Retrieves the maximum value in a column. 
    SELECT MAX(salary) FROM EMPLOYEES;
  • **SUM**: Calculates the total sum of a numeric column. 
    SELECT SUM(salary) FROM EMPLOYEES;
  • **COUNT**: Counts the number of rows in a column. 
    SELECT COUNT(id) FROM EMPLOYEES;
  • **DISTINCT**: Retrieves distinct values from a column. 
    SELECT DISTINCT(name) FROM EMPLOYEES;
  • **ROUND**: Rounds a numeric column to a specified number of decimal places. 
    SELECT ROUND(Marks, 2) FROM STUDENTS;

String Functions (String Handling Operations)

  • **LENGTH**: Returns the length of a string. 
    SELECT LEN('APPLE');
  • **REVERSE**: Reverses a string. 
    SELECT REVERSE('APPLE');
  • **REPLICATE**: Repeats a string a specified number of times. 
    SELECT REPLICATE('APPLE', 5);
  • **UPPERCASE**: Converts a string to uppercase. 
    SELECT UPPER('apple');
  • **LOWERCASE**: Converts a string to lowercase. 
    SELECT LOWER('APPLE');
  • **CONCATENATION**: Concatenates two or more strings. 
    SELECT CONCAT('NE', 'PA', 'L');
  • **LEFTMOST CHARACTER**: Retrieves the leftmost characters of a string. 
    SELECT LEFT('APPLE', 3);
  • **RIGHTMOST CHARACTER**: Retrieves the rightmost characters of a string. 
    SELECT RIGHT('APPLE', 3);

Other SQL Operations

  • **IN**: Checks if a value matches any value in a list. 
    SELECT ADDRESS FROM STUDENTS WHERE ADDRESS IN ('KTM', 'BKT', 'PTN');
  • **BETWEEN**: Retrieves values within a given range. 
    SELECT SALARY FROM EMPLOYEES WHERE SALARY BETWEEN 30000 AND 40000;
  • **LIKE** (Percentage %): Matches patterns in strings. 
    SELECT NAME FROM STUDENTS WHERE NAME LIKE '%a';
  • **LIKE** (Underscore _): Matches a single character in a string. 
    SELECT NAME FROM STUDENTS WHERE NAME LIKE '_b%';
  • **HAVING**: Filters results after an aggregate function is applied. 
    SELECT NAME FROM STUDENTS WHERE GRADE = 'A' HAVING MARKS > 800;
